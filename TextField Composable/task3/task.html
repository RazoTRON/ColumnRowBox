Search field # Application

Below is a composable SearchField you need to create. Users should only be able to enter digits in this field. In addition, this composable must have two icons located on the left and right side of the text field. You can use any icons, or take the ones used in the example:

Icon(imageVector = Icons.Default.Phone, contentDescription = "CHANGE DESCRIPTION")

Icon(imageVector = Icons.Default.Search, contentDescription = "CHANGE DESCRIPTION")

[ALERT-warning]Make sure to change the description of the contentDescription parameter. To pass the test, the description must match the TextField parameter that uses this icon.[/ALERT]

Example:

[ALERT-warning]To pass tests, DO NOT import androidx.compose.material.TextField or OutlinedTextField .[/ALERT]
